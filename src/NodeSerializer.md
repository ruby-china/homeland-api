# NodeSerializer

节点详情

## attributes

* **id** [Integer] 编号
* **name** [String] 节点名称
* **summary** [String] 简介，Markdown 格式
* **section_id** [Integer] 大类别编号
* **section_name** [String] 大类别名称
* **topics_count** [Integer] 话题数量
* sort **Integer** 排序优先级
* **updated_at** [DateTime] 更新时间
